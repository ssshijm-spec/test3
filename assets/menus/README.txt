[ 음식 사진 넣는 방법 ]

이 폴더에 메뉴별 사진을 넣으면 게임 카드에 자동으로 표시됩니다.
사진이 없어도 게임은 색상+이모지 플레이스홀더 카드로 정상 동작합니다.

1) 파일 이름 = 프로젝트 루트 config.js 의 각 메뉴 "slug" 값과 동일하게 저장.
     예) 제육볶음 -> jeyuk-bokkeum.png
         김치찌개 -> kimchi-jjigae.png
         돈까스   -> donkatsu.png
   지원 확장자: .png / .jpg / .jpeg / .webp  (자동 탐색, png 우선)

2) 권장 이미지: 정사각형(512x512 권장, 최소 256x256), 음식이 가운데,
   배경 단순. 게임이 자동으로 픽셀아트 톤(다운스케일 + 팔레트 스냅)으로 변환합니다.

3) 사진을 넣거나 바꾼 뒤 브라우저 새로고침(F5) 하면 즉시 반영됩니다.

주의) 로컬 파일(file://)로 열면 브라우저 보안정책 때문에 사진이 카드에 안 그려질 수 있습니다.
      반드시 프로젝트 루트의 start.bat (로컬 서버)로 실행하세요.

전체 메뉴/슬러그 목록은 config.js 의 FOODS 배열에서 확인/수정할 수 있습니다.
